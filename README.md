# Noonsaegim

사진을 제출하면 객체를 인식하여 영어단어와 의미를 알려주는 인공지능 앱입니다.

알림을 설정해서 주기적으로 들여다 보며 쉽게 영어 단어를 외울 수 있게 하기 위해서 기획하게 되었습니다.
개인 단어장도 만들수 있고 원하는 리스트의 데이터를 MP3, PDF 형식의 파일로 변환해 다운받거나 공유할 수도 있습니다.
